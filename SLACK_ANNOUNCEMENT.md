I have an initial Companion v3 module ready for EZCast Pro / Actions Micro AM8270 / PWAY/PWAYTek ProAV POE IP KVM extenders:

`companion-module-EZCAST-PRO-DT214M-AM8270-KVM`

It controls ER02 / ProAVRx receivers over the CMS JSON-RPC API, supports RX channel switching, RX/TX naming, guarded TX ID changes, subnet discovery, current-source variables, HDMI/connection feedbacks, and presets for channels `01` through `99`.

Search/listing terms for the hardware include:

`150m hdmi POE KVM extender over ip many-to-many`

The v0.1.0 release package is:

`ezcast-pro-dt214m-am8270-kvm-0.1.0.tgz`
